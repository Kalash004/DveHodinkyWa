  • Endpoint Descriptions
    • GET /api/messages
      • Get all messages in db. Messages are either groupChat messages or dms.
    • GET /api/messagesUser?user=(username/email)
      • Get all messages posted by a specific user
    • GET /api/messagesGroup
      • Get all messages of a specific groupChat.
    • GET /api/messagesWord?word=(word)
      • Get all messages containing specified text.
